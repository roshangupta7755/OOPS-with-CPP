## 🚀 About This Repository

This repository contains **Object-Oriented Programming (OOP)** examples written in **C++**, designed for students learning OOP concepts such as:

* Classes & Objects
* Constructors & Destructors
* Inheritance
* Polymorphism
* Encapsulation
* Abstraction
* File Handling

Everything is arranged neatly for easy learning.
